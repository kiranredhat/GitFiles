1) To see all commit log
git log
2) git log -p
3) To see only commit message
git log --oneline
4) commit last two log
 git log -2
5) To grep paticular commit as per commit text of commit 
git log -i --grep="Text of commit"
6) To get commit as per text in code
 git log -i -G"Text in code"
OR
git log -i -p -G "Text in code"
7) git log--stat
8) To get log as per author
git log -i --author="kiranredhat"
OR
git log -i --author="kiran.gaikwad45@gmail.com"
